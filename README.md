# FreshTurn Platform

FreshTurn is a browser-based restaurant inventory management and surplus food marketplace prototype.

## What it now includes

- Demo sign-in for `Admin`, `Manager`, and `Customer`
- Multi-branch inventory tracking
- Per-branch closing mode toggles
- Automatic markdown suggestions for food at risk of waste
- Customer reservation flow for discounted meals
- Reservation board and branch-level impact analytics
- Persistent local browser storage so the demo keeps its state after refresh

## Run it

Open `index.html` in a browser.

5. Open the Reservations and Insights views to track the result.

## Note

This version is self-contained because no Node.js or Python runtime was available in the workspace. The app behaves like a product prototype using persistent browser storage and can later be migrated to a server-backed stack.
